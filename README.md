# 📚 AI-Powered Children's Story & Book App

An intelligent web application that allows children to **generate, read, and interact with personalized stories** using Artificial Intelligence.
This project combines modern web technologies with AI to create an engaging, educational, and fun storytelling experience.

---

## 🚀 Features

* 🤖 **AI Story Generation**
  Generate unique children’s stories based on user input (theme, idea, age).

* 🎯 **Personalized Content**
  Stories can be tailored to the child’s preferences, making them more engaging.

* 📚 **Digital Story Library**
  Browse and manage generated stories in one place.

* 🎨 **Modern UI/UX**
  Clean, responsive, and child-friendly interface.

* 🔐 **Authentication System**
  Secure login and user session management.

* 📱 **Responsive Design**
  Works smoothly on mobile, tablet, and desktop devices.

---

## 🛠️ Tech Stack

* **Frontend:** Next.js, TypeScript
* **Styling:** Tailwind CSS
* **AI Integration:** OpenAI API (or any AI text generation service)
* **Backend:** API routes / external services
* **Deployment:** Vercel (recommended)

## ⚙️ Installation & Setup

1. **Clone the repository**

```bash
git clone https://github.com/your-username/AI-Powered_Children-s_Story_-_Book_App.git
```

2. **Navigate to the project folder**

```bash
cd AI-Powered_Children-s_Story_-_Book_App
```

3. **Install dependencies**

```bash
npm install
```

4. **Add environment variables**
   Create a `.env.local` file and add:

```
OPENAI_API_KEY=your_api_key_here
```

5. **Run the development server**

```bash
npm run dev
```

6. Open in browser:

```
http://localhost:3000
```
